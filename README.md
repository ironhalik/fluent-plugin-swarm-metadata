## Installation

### RubyGems

```
$ gem install fluent-plugin-swarm-metadata
```

### Bundler

Add following line to your Gemfile:

```ruby
gem "fluent-plugin-swarm-metadata"
```

And then execute:

```
$ bundle
```

## Configuration

You can generate configuration template:

```
$ fluent-plugin-config-format filter swarm-metadata
```
